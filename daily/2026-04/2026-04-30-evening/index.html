🚀 英伟达发布全模态Agent模型，AI竞争转向"让模型干活"

4月29日深夜，英伟达重磅开源Nemotron 3 Nano Omni——30B-A3B MoE混合专家架构的全模态大模型，原生支持文本、图像、视频和音频输入，256K超长上下文。这是首个将视觉、语音、文本能力整合进单一MoE模型的产品级方案：推理吞吐量比同尺寸开源模型高9倍，在OCRBench-V2、WorldSense、DailyOmni等基准上领跑。模型采用7阶段渐进式训练（从视觉对齐到256K超长上下文），并引入Conv3D视频压缩和Efficient Video Sampling两项token缩减技术，在几乎不损失准确率的前提下将时延降低33%。英伟达同步开出了BF16/FP8/FP4三种精度权重和训练数据。

全模态Agent开源 来源：NVIDIA · 新浪财经 · arXiv 2604.24954

🏦 蚂蚁集团开源万亿参数百灵Ling-2.6-1T

4月30日，蚂蚁百灵大模型宣布开源Ling-2.6-1T。采用MLA与Linear Attention的Hybrid架构，核心目标不是堆参数而是系统性优化智效比——通过抑制"过程冗余"的强化奖励策略，压缩输出成本的同时保持强综合智能。在AIME26、SWE-bench Verified、BFCL-V4、TAU2-Bench等执行类基准上达到开源SOTA。官方强调"让万亿模型真正进入开发者和企业的生产工作流"。

🐙 GitHub 亮点

SWE-agent/mini-swe-agent ★3,614 — 100行代码的AI编程Agent

普林斯顿+斯坦福团队打造的极简AI软件工程agent。核心理念——工具只有一个：bash。不需要function calling，不需要复杂配置。每个动作用subprocess.run执行，完全独立，线性历史。在SWE-bench Verified上达到>74%，已被Meta、NVIDIA、IBM等采用。最新v2.2.8支持Gemini 3 Pro、GPT-5等模型，随机切换模型甚至能提升性能。代表了"让模型本身成为Agent核心，而不是脚手架成为核心"的设计哲学。

PythonMIT 40位贡献者 · 56个Release

GitHub Agentic Workflows ★4,000 — 官方Agent CI/CD平台

GitHub官方推出的Agent自动化工作流平台，通过简单的Markdown文件定义，让仓库每天早上自动收到改进PR。已支持Copilot、Claude、Codex和OpenCode四种AI引擎，v0.68.6版本新增bare模式（跳过AGENTS.md上下文加载）。安全优先设计，运行在GitHub Actions沙箱中。

Go GitHub官方 · 开源

open-gitagent/gitagent ★281 — Git原生AI Agent框架

将Agent的身份、规则、记忆、工具和Skills全部以文件形式管理在Git仓库中，实现Agent即代码。TypeScript实现，MIT协议，已发布10个Release。

TypeScriptMIT 3位贡献者

面向GitHub Copilot的多Agent编排系统，结构化工作流（指挥→规划→实施→审查→完成），11个核心Agent + 5个翻译Agent。提供可复用技能和验证工具，旨在让GitHub Copilot从单人助理升级为多角色工程团队。
